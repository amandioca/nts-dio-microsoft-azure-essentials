### **O que é _Jump Server_ (_Bastian host_)?**

→ É um servidor controle de acesso privilegiado e gerenciamento de conexões a outros servidores internos, comumente usado para proteger ambientes corporativos.

→ "_Bastian_" (Bastião) é uma torre fortificada de um castelo ou muralha.

* Centraliza acesso
* Permite ou Barra entrada
* Aplica Regras: _MFA, SSH, VPN, IP fixo, firewall..._
* Audita acesso
* Como um "**porteiro digital**"

→ Serviços marcados com "Prévia" não possuem `SLA*` ou seja, sem garantias da Azure.

#### `*SLA` (_Serviço Level Agreement_): 
Contrato formal que define padrões de qualidade, métricas de desempenho e prazos de entrega.
- Métricas de desempenho
- Responsabilidades Claras 
- Prazos (Deadlines)
- Penalidades / Recompensa
