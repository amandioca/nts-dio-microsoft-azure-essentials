*O que é Jump Server (Bastian host)? *
→ É um servidor controle de acesso privilegiado e gerenciamento de conexões a outros servidores internos, comumente usado para proteger ambientes corporativos.
- "Bastian" (Bastião) é uma torre fortificada de um castelo ou muralha.
* Centraliza acesso: MFA, SSH, VPN, IP fixo... 
* Permite ou Barra entrada
* Aplica Regras
* Audita acesso
- Como um "porteiro digital"

Serviço marcados com "Prévia" não possuem SLA* ou seja, sem garantias da Azure.
*SLA (Serviço Level Agreement): Contrato formal que define padrões de qualidade, métricas de desempenho e prazos de entrega.
- Métricas de desempenho
- Responsabilidades Claras 
- Prazos (Desalinhes)
- Penalidades / Recompensa
